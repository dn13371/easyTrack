# EasyTack

## Documentation: 
https://dn13371.github.io/easyTrack/

## Getting started 

### Clone repo

```bash

git clone https://github.com/dn13371/easyTrack.git

```

### Create venv

```bash

python -m venv venv 
source venv/bin/activate


```

### Install requierements

```bash

pip install -r requirements.txt


```

### Initialized Database 

```bash

flask init-db

```

### Run easyTrack

```bash

flask run

```
